Команды
RW - рисование ж.д. методом рельсы-рельсы, шпалы-шпалы
Zebra - рисование ж.д. полилинией
Zebra2 - рисование ж.д. черной штриховкой и пропуском
Zebra3 - рисование ж.д. черной и белой штриховкой
Zebra4 - рисование ж.д. одной черной штриховкой



Список файлов

zebra.LSP        - файл с кодом
zebra.lin           - файл типа линии Zebra
mip.shx            - сами формы (скомпилированы)
mip.shp            - описание форм (некомпилированное)
Zebra.BMP       - рисунок для кнопки
ZEBRA2.BMP   - рисунок для кнопки
zebra.mns        - меню для подгрузки в AutoCAD
readme.txt        - этот файл

Как пользоваться
1. Поместите файлы в папку, прописанную в путе доступа к вспомогательным файлам
   (Сервис->Настройка->Файлы->Путь доступа к вспомогательным файлам   Tools->Options->Files->Support File Search Path)
2. командой _menuload подгрузить файл zebra.mns, активизировать панель Zebra
