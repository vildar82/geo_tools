Редактировать параметры текущего профиля

Позволяет редактировать некоторые параметры текущего профиля

Команда: profile_edit_properti_curent_basement

Текущий профиль:[имя текущего профиля]

В появившемся окне меняет доступные параметры на нужные, при чем изменение параметров профиля не затронет уже существующие объекты профиля, а при создании новых (например подписи отметок) будут использоваться новые настройки. Для чего какая настройка можно подробнее посмотрать в описании команды Создать профиль.
