Оформление листа

Оформление нового листа и видового экрана по блоку вспомогательной рамки.

Работа с программой предпологает следующую процедуру. Сначала пользователь разбивает, подготавливаемую к печати трассу, на листы с помощью вспомогательных рамок показывающих полезную площадь форматного листа за вычетом штампа (для генерации вспомогательных рамок используется Менеджер форматов).

В инструкциях есть статья описывающая взаимодействие программ.

Затем запускаем программу с помощью которой имеем возможность преобразовать любую из вспомогательных рамок в оформленый лист с заполненым штампом.

Команда: draw_layout_and_viewport_be_format_block

Текущие настройки: Вернуться в модель: Да; Тип вставляемого штампа: текущий штамп (в этой строке указаны текущие настройки)

Название листа [Настройки] <автоимя следующего листа>:

Вводит название нового листа или выбирает Настройки

[Вернуться в модель/Тип штампа]<продолжить>:

Выбирает что хочется настроить Тип штампа

Тип вставляемого штампа[Авто/ГОСТ/Большой/Маленький/Нет]<текущий штамп>:

Выбирает тип вставляемого с листом штампа, режим Авто подразумевает автоопределение нужного штампа исходя из того какой штамп был выбран при создании вспомогательной рамки. Вставляется текущий штамп! настройка текущего штампа производится в программе Штамп редактор и Штамп редактор ГОСТ.

или Вернуться в модель

Вернуться в модель[Да/Нет]<Да>:

Выбирает Да если после создания листа необходимо вернуться в модель (например чтобы сразу указать следующий лист) или Нет чтобы остаться в листе

Выбери блок вспомогательной рамки:

Указывает блок вспомогательной рамки
