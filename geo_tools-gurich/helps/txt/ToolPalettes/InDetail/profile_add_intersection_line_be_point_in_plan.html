Команда: profile_add_intersection_line_be_point_in_plan

Текущий профиль:[имя текущего профиля]

Укажи точку на плане:

Указывает точку пересечения с комуникащией на плане

Укажи объект типа текст<ввести>:

Указывает текст содержимое которого будет использоваться на профиле для подписи пересечки или пустой ввод (Enter) для того что бы ввести текст с клавиатуры.

Введи текст пересечки:

Вводит текст пересечки с клавиатуры

Вводит отметку пересечки с клавиатуры или пустой ввод (Enter), при указаной отметке она будет обозначена треугольным маркером, если отметку не указывать вертикальная линия будет продолжена за приделы профиля.
