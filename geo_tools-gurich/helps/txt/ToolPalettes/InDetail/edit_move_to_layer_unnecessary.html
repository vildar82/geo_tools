Лишний пикет

Меняет слой выбранных объектов на указанный

Команда: edit_move_to_layer_unnecessary

Введи имя целевого слоя [ВЫБРАТЬ/ТЕКУЩИЙ]<последний используемый>:

Вводит имя слоя в который будут перемещаться объекты, если такого слоя не существует будет предложено его создать. Enter (пустой ввод) означает выбор использованного ранее слоя. Если выбрать пункт меню ТЕКУЩИЙ от целевым будет принят текущий слой. При выборе пункта ВЫБРАТЬ будет выведено диалоговое окно выбора слоя из существующих.

Выберите объекты:

Выбирает объекты для первода на выбранный слой
