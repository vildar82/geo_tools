Подписать горизонталь (не Civil)

Подписывает горизонталь: дугу, полилинию, сплайн. Для тех кто не пользуется Civil.

Команда: draw_elevation_contour_line

Выбери горизонталь (не Civil):

Указывает объект типа: отрезок, дугу, полилинию, сплайн.

Укажи точку вверх по склону[Настройки]:

Указывает точку "в гору" для правильной ориентации отметки. Или выбирект пункт Настройки тогда получает диалоговое окно настроек.

Объекты автокада изображающие горизонтали могут иметь отметку (положение по оси Z) а могут и не иметь. Могут иметь и неправильную отметку. Для урегулирования ситуации существует первая опция "Запрашивать отметку" в зависимости от выбранного пункта выпадающего списка программа будет запрашивать отметку:

Только когда 0 - только тогда когда отметка указанного объекта будет равна нулю будет запрошена отметка у пользователя с клавиатуры вот так вот:

Введи отметку горизонтали:

Всегда - полностью игнорируя отметки объектов программа каждый раз запрашивает пользователя.

Никогда - пользователь программой не беспокоится и все горизонтали подписываются без запросов с клавиатуры.

При разном сечении рельефа есть необходимость подписывать горизонтали с разной точностью: для целых горизонталей "200" для дробных "200.5". В выпадающем списке Точность отметки можно выбрать необходимую точность. Умная точность отметок горизонталей позволяет подписывать с нужной точностью наиболее распространенные сечения рельефа: 200.000 подпишется как 200, 200.500 как 200.5, 200.250 как 200.25, 200.750 как 200.75. Все остальные значения будут подписаны до милиметров.
