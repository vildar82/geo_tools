Штриховка (отрисовка площадных условных знаков)

Заполняет замкнутую полилинию соответствующим условным знаком

Программа не моя

Нашел ее где то на просторах Интернета, сейчас уже не помню где. Нужна для заполнения блоками замкнутой полилинии. В оригенале есть возможность определения зоны штриховки по точкам указываемым точкам. Немного подправил прогу (в основном это связано с порядком вопросов задаваемых программой) чтобы ее можно было использовать для автоответа на вопросы на кнопках.

Пример макроса(содержимое кнопки):

^C^Cfil;Y;Штриховка Газон;P;10;5

символ ";" соответствует нажатию клавиши Enter

Команда: fil

Будем заполнять контур в шахматном порядке? (Yes/No) <N> :

Y

Введи имя блока для штриховки :

Штриховка Газон

Choose object type [R]ectang or [P]olygon <R> :

P

Введи рассояние между колонками в мм плана (для шахматного порядка увелич это число в двое) :

10

Введи рассояние между строками в мм плана :

5

Выбери ЗАМКНУТУЮ полилинию для штриховки:

указывает заранее построеную замкнутую полилинию
