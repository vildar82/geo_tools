Экспорт точек в файл координат

Экспортирует выбранные точки в файл, форматов Nikon PNEZD,CDR33,GSI

Позволяет записать выбраные cogo точки в файл выбранного формата. Еще один вариант перемещения координат точек в прибор.

Команда: export_cogo_points_to_be_file

Выберите объекты:

указывает объекты типа cogo точка

Поле Номер соответствует номеру точки в создаваемом файле.

Описание соответственно описанию.

кнопки номер, имя и описание вставляют в текущую строку конструкцию вида [P], [N] или [D], которая будет заменена на номер, имя или описание данной cogo-точки. При не заполненном поле Номер точки будутпронумерованы по порядку в 1. Следует помнить, что при загрузке в Nikon в одном проекте не может содержаться двух точек с одинаковым номером.

стандартное окно сохранить как... для сохранения выходного файла с расширением.

Имя файла по умолчанию формируется [имя чертежа]_[имя слоя первой точки] Каталогом для сохранения файла по умолчанию предлагается расположение открытого файла если файл еще не был сохранен будет выбрана папка Мои документы текущего пользователя.

выбирает место и имя сохраняемого файла
