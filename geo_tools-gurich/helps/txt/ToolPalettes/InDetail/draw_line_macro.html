Макросы выбора типа линии, толщины и инструмента

Среди инструментов много кнопок выстрой смены текущего типа линий с последующим вызовом команды рисования. Устроено это так:

Каждая кнопка это набор макрокоманд которые и выполняют всю настройку. Среди этих команд есть и мои.

Линии.

Самый простой тип.Пример макроса:

^C^C_format_check_line_typ;ЛИНИЯ_2-1;_line;

символ ";" соответствует нажатию клавиши Enter

Команда:format_check_line_typ

Введи имя типа линии:

ЛИНИЯ_2-1

Команда:_line

Если указаного типа нет в рисунке он будет подгружен из внешнего .lin файла

Со сплайнами абсолютно так же как и с линиями отличие только в конце, вместо стандартной команды отрезов вызывается сплайн.

Полилинии.

Пример макроса:

^C^C_format_check_line_typ;ЛИНИЯ_2-1;format_poly_width;0;_pLine;

символ ";" соответствует нажатию клавиши Enter

Команда:format_check_line_typ

Введи имя типа линии:

ЛИНИЯ_2-1

Команда:format_poly_width

Введи толщину полилинии в милиметрах:

0

Команда:_pline

Если указаного типа нет в рисунке он будет подгружен из внешнего .lin файла

Программа вычисляет какой должна быть толцина полилинии чтобы при печати в текущем масштабе аннотаций на бумаге получилась введеная толщина.
