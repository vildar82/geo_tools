Сделать тип линии объекта текущим

Установка текущим типа линии, соответствующего выбранному объекту

Команда: format-put-sel-linetype

Выберите объект, тип линии которого должен стать текущим:

указывает объект с которого нужно взять тип линии, как вариант можно указать объект заранее и вызвать программу, тогда запроса объекта не последует.

Тип линии [тип линии] установлен текущим.

Команда практически идентична автокадовской "Сделать слой объекта текущим" только работает с типом линии.
