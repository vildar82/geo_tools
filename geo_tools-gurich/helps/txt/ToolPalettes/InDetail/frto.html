Замена объектов объектом

Команда позволяет заменить каждый из набора объектов на какой то другой

Програмка слита с Интернета, дело было давно, где взял не помню.

Команда: frto

+++ Select source object ->

указывает объект который послужит образцом

+++ Select destination objects and press Enter <-

Выберите объекты:

указывает объекты для замены или наложения

Remove destination object [Yes/No] <No>:

отвечает Yes если необходимо удалить объекты и на их место скопировать объект образец или отвечает No если удалять объекты не требуется и объект образец скопируется поверх них.
